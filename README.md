# Age & Gender Detection (OpenCV)

## Overview
This project detects faces and predicts **age group** and **gender** using OpenCV DNN and pretrained models.

## Tech Stack
- Python
- OpenCV (DNN)
- Pretrained age + gender models

## Folder Structure
- `src/` → code
- `models/` → pretrained model files
- `images/` → sample outputs/screenshots

## How to Run
```bash
pip install -r requirements.txt
python src/main.py
